Usage

Buttons come in 4 different variations. They can be Raised (by default), Outlined, Flat or Floating. 

<button class="btn"><span>default</span></button>

Outline Buttons

Outline buttons have no background and instead they add a border from the current color. 

<button class="btn btn--outlined"><span>outlined</span></button>
 <button class="btn btn--outlined text-secondary"><span>outlined</span></button>

Floating Buttons

To make a FAB (Floating action button), add the class btn-floating. FAB's are round and they usually contain an icon. 

<button class="btn btn--floating">
    <i class="material-icons icon">create</i>
 </button>
 <button class="btn bg-secondary btn--floating">
    <i class="material-icons icon">call</i>
 </button>

Shaped Buttons

<button class="btn btn--raised rounded">...</button>
 <button class="btn bg-secondary btn--raised rounded-semi">...</button>
 <button class="btn bg-secondary btn--raised rounded-semi-reverse">...</button>

Buttons with icon

You can easily add any icon in the button. 

<button class="btn btn--raised">
    <span>analyze</span>
    <i class="material-icons icon space-left--sm">send</i>
 </button>

Button with different sizes

You can make buttons small and large by adding btn-sm and btn-lg classes. 

<button class="btn btn--sm">analyze</button>
 <button class="btn btn--lg">analyze</button>

Block Buttons

Block buttons extend the full available width. 

<button class="btn btn--block">analyze</button>

Classes Details

CSS Class Explanation
btn Sets up basic properties for button
btn--outlined Removes background and instead adds border from the current color
btn--floating Floating action button
btn--block Full width button
