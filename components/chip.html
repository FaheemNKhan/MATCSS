Usage

Chips can be used to show information, such as a contacts or they can be used to create different items.

Simple Chip
Another Chip 
<div class="chip">
    <span class="chip-text">Simple Chip</span>
 </div>  
...

Chip with Image/Icon

You can use any image/icon inside chip.

With Image
mode_comment With Icon
Close Icon close
Another Chip 
<div class="chip">
    <img src="imageurl" alt="image alt text" />
    <span class="chip-text">With Image</span>
 </div>  
 <div class="chip">
    <i class="material-icons icon">mode_comment</i>
    <span class="chip-text">With Icon</span>
 </div>  
 <div class="chip">
    <span class="chip-text">Chip Text</span>
    <span class="chip-close material-icons icon">close</span>
 </div>

Classes Details

CSS Class Explanation
chip Container for chip content
