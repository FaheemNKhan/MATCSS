Avatar Sizes

You can specify different sizes for avatars: medium (default), small or large. 


<div class="avatar avatar--sm">..</div>
 <div class="avatar avatar--lg">..</div>
Classes Details
CSS Class Explanation
avatar Avatar container, required
avatar--sm Small Size Avatar
avatar--lg Large Size Avatar
