Usage

Bottom navigation is used to easily switch between top-level views.

Any bottom navigation item with class is-active will be activated.

  • wifi
  • 3d_rotation
  • room
  • 09:00
house Home
archive Inbox
touch_app Extra
settings Settings 
<nav class="bottom-nav">
    <div class="bottom-nav-item">
        <span class="bottom-nav-icon material-icons icon">house</span>
        <span class="bottom-nav-text">Home</span>
    </div>
    <div class="bottom-nav-item is-active">
        <span class="bottom-nav-icon material-icons icon">archive</span>
        <span class="bottom-nav-text">Inbox</span>
    </div>
    <div class="bottom-nav-item">
        <span class="bottom-nav-icon material-icons icon">touch_app</span>
        <span class="bottom-nav-text">Extra</span>
    </div>
    <div class="bottom-nav-item">
        <span class="bottom-nav-icon material-icons icon">settings</span>
        <span class="bottom-nav-text">Settings</span>
    </div>
</nav>

Bottom Navigation with Shift Animation

Use the bottom-nav-shifting to create a shift effect.

This will hide the text from other bottom-navigation-item's.

  • wifi
  • 3d_rotation
  • room
  • 09:00
house Home
archive Inbox
touch_app Extra
settings Settings 
<nav class="bottom-nav bottom-nav--shifting bg-dark">
    <div class="bottom-nav-item">
        <span class="bottom-nav-icon material-icons icon">house</span>
        <span class="bottom-nav-text">Home</span>
    </div>
    <div class="bottom-nav-item is-active">
        <span class="bottom-nav-icon material-icons icon">archive</span>
        <span class="bottom-nav-text">Inbox</span>
    </div>
    <div class="bottom-nav-item">
        <span class="bottom-nav-icon material-icons icon">touch_app</span>
        <span class="bottom-nav-text">Extra</span>
    </div>
    <div class="bottom-nav-item">
        <span class="bottom-nav-icon material-icons icon">settings</span>
        <span class="bottom-nav-text">Settings</span>
    </div>
 </nav>

Bottom Navigation with Icons

If you don't want text label inside bottom navigation item, use bottom-nav--default.

  • 3d_rotation
  • room
  • 09:00
house
archive 2
touch_app
settings 
<nav class="bottom-nav bottom-nav--default">
    <div class="bottom-nav-item">
        <i class="bottom-nav-icon material-icons icon">home</i>
    </div>
    <div class="bottom-nav-item">
        <i class="bottom-nav-icon material-icons icon">archive
            <span class="badge">2<span>
        </i>
    </div>
    <div class="bottom-nav-item is-active">
        <i class="bottom-nav-icon material-icons icon">touch_app</i>
    </div>
    <div class="bottom-nav-item">
        <i class="bottom-nav-icon material-icons icon">settings</i>
    </div>
 </nav>
Clipped Bottom Navigation

For fixed bottom navigation, use clipped and clipped--bottom modifiers. 

<nav class="bottom-nav clipped clipped--bottom">
    ...
 </nav>

Classes Details

CSS Class Explanation
bottom-nav Acts as a container
bottom-nav--shifting Applies shifting animation
bottom-nav--default Normalizes bottom navigation item
bottom-nav-item Container for icon and text
is-active Highlights the selected item
bottom-nav-text Contains text
bottom-nav-icon Contains icon
