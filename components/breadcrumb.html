Breadcrumb

  • Usage
  • Breadcrumb with custom Separator
  • CSS Classes explanation

Usage

Breadcrumb is a secondary navigation that shows user’s location in a site hierarchy.

Link1 Link2 Link3 

<nav class="breadcrumb">
    <a href="" class="breadcrumb-link">Link1</a>
    <a href="" class="breadcrumb-link">Link2</a>
    <a href="" class="breadcrumb-link is-active">Link3</a>
 </nav>

Breadcrumb with custom Separator

To remove default separator, use breadcrumb--custom class.

Link1
arrow_right_alt
Link2
arrow_right_alt Link2 

<nav class="breadcrumb breadcrumb--custom">
    <a href="" class="breadcrumb-link">
        <span>Link1</span>
        <i class="material-icons icon">arrow_right_alt</i>
    </a>
    <a href="" class="breadcrumb-link">
        <span>Link2</span>
        <i class="material-icons icon">arrow_right_alt</i>
    </a>
    <a href="" class="breadcrumb-link is-active">Link3</a>
 </nav>
Classes Details
CSS Class Explanation
breadcrumb Breadcrumb container
breadcrumb-link Breadcrumb's link styling.
is-active Breadcrumb's link active state
breadcrumb--custom Removes default separator
