Usage

A Dialog is a small window that communicates information to the user and prompts them for a response.

Dialog structure has dialog surface, overlay and action buttons, defined by dialog-surface, overlay which is optional and dialog-actions, respectively.

.overlay component must be placed inside the dialog for proper functionality.

Dialog Title

Dialogs inform users about a specific task 

<button type="button" class="btn" id="btn-dialog-1" 
            onclick="Dialog('#dialog1')">open dialog</button>

 <div class="dialog" id="dialog1">
    <div class="overlay" onclick="Dialog('#dialog1')"></div>
    <div class="dialog-surface">
        <h2 class="dialog-title">Dialog Title</h2>
        <p>Dialog Text</p>
        <div class="dialog-actions">
            <button type="button" class="btn">action 1</button>
        </div>
    </div>
 </div>

You can replace onclick with addEventListener. A sample javascript implementation is as follows: 

var Dialog = (function(doc){
    return function(dialogId){
        doc.querySelector(dialogId).classList.toggle('is-active');
        doc.querySelector(dialogId).querySelector('.overlay').classList.toggle('is-active');
        doc.querySelector('html').classList.toggle('no-scroll'); // to stop background scrolling
    }
 })(document);

Classes Details
CSS Class Explanation
dialog Required ! Acts as a container for all dialog content
overlay Creates dialog overlay, can be used to close dialog
dialog-surface Required ! Contains dialog title, dialog content and dialog actions
dialog--width-auto Positions dialog to take only required width
dialog--top-sheet Converts dialog into a top sheet
dialog--bottom-sheet Converts dialog into a bottom sheet
