Usage

A Card is a standalone component that contain content and actions about a single element.

There are 3 components of card: card-section which is the only required one, card-media which contains image and card-actions which contains action buttons.

Classes Details
CSS Class Explanation
card Acts as a container for card content
card-section Separates card content from each other
card-media Contains card image/video
card-media-text Title and subtitle of card's media
card-actions Contains card button and icons
