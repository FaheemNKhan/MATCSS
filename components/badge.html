Usage

Badges can be used to draw attention to a specific element.

A Badge is an inline element that appears in another element as notifier or indicator, for example, to notify a user about a new message or to display items count in the shopping cart, etc.

By default, the secondary theme color is used as badge background.

group 12 add_shopping_cart 18 

<i class="material-icons icon">group<span class="badge">12</span></i>
 <i class="material-icons icon">add_shopping_cart<span class="badge">18</span></i>

Classes Details
CSS Class Explanation
badge Container, required!
badge--tile Badge tile, optional
badge--inline Badge position to inline, optional
badge--top-left Badge at the top left corner
badge--bottom-left Badge at the bottom left corner
badge--bottom-right Badge at the bottom right corner
