Usage

The navigation drawer slides in from the left or right and contains the navigation destinations for your app.

There are 3 components of navigation drawer: drawer which is the only required element, drawer-overlay which can be used to close navigation drawer and drawer-header which contains navigation drawer title or image.

Navigation drawers are closed by default, you just have to toggle is-active class on drawer and the overlay (optional).

Persistent Navigation Drawer

Follow these steps to convert a regular or permanent drawer into a persistent drawer.

  • Toggle drawer-offset-left and drawer--permanent classes on screens > medium.
  • Toggle is-active class on both drawer and overlay on screens < medium. 
    •                 var Drawer = (function(doc){
                    return function(drawerId, overlayId){
                    if(window.innerWidth > 960){
                    doc.querySelector('main').classList.toggle('drawer-offset-left');
                    doc.querySelector(drawerId).classList.toggle('drawer--permanent');
                    } else {
                    doc.querySelector(drawerId).classList.toggle('is-active');
                    doc.querySelector(overlayId).classList.toggle('is-active');
                    }
                    }
                    })(document);

    Classes Details

    CSS Class Explanation
    drawer Required ! Acts as a container for all drawer content
    drawer-overlay Creates drawer overlay, can be used to closed drawer
    drawer-title Drawer title styling
    drawer-subtitle Drawer subtitle styling
    drawer-header Contains user information
    drawer-header-sm Creates a small header
    drawer-header-lg Creates a large header
    active Activates drawer
    drawer-right-align Positions Navigation drawer to the right side of screen
    drawer-fixed Applies fixed position to the navigation drawer
