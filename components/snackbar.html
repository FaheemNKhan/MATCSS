  • Snackbar variations

Usage

Snackbar is a simple popup notification for quick messages. It consists of 2 components: snackbar, snackbar-surface, and snackbar-message, snackbar-actions which are optional.

Video download has failed.
close 
<aside class="snackbar">
    <div class="snackbar-surface">
        <div class="snackbar-message">Video download has failed.</div>
        <div class="snackbar-actions">
            <button type="button" class="btn">retry</button>
            <i class="material-icons icon">close</i>
        </div>
    </div>
 </aside>

Snackbar variations

  • To show snackbar, just add the class is-active.
  • Even though snackbar will be hidden again after 5 seconds (by default), it won't be removed from normal page flow, you should remove is-active class once the snackbar is hidden.

    <div class="snackbar-surface">
        <div class="snackbar-message">Connection timeout. Retrying after 10 seconds again to complete download.</div>

            <button type="button" class="btn">retry</button>
            <i class="material-icons icon">close</i>
        </div>
    </div>
 </aside>

Persistent Snackbar

Use snackbar--persistent to make the snackbar persistent. 

<aside class="snackbar snackbar--persistent">
    <div class="snackbar-surface">
        <div class="snackbar-message">Snackbar with infinite duration.</div>
        <div class="snackbar-actions">
            <button type="button" class="btn">retry</button>
            <i class="material-icons icon">close</i>
        </div>
    </div>
</aside>
Classes Details
CSS Class Explanation
snackbar Acts as a wrapper for the underlying content
snackbar-surface Snackbar surface, contains snackbar message and actions
snackbar-message Contains snackbar slogan text
snackbar-actions Contains buttons/icons
snackbar--start Start aligned snackbar (Desktop Only)
snackbar--persistent Snackbar with no timeout
snackbar--vertical Stacked snackbar
is-active Activates snackbar
