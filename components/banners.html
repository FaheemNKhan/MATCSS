Banners

  • Usage
  • Banners with Actions
  • CSS Classes explanation

Usage

A banner can be used to display a message and related optional actions.

wifi_off There is no network connection. Please try later! 

<div class="banner">
    <div class="banner-container">
        <div class="banner-content">
            <i class="material-icons icon">wifi_off</i>
            <span>There is no network connection. Please try later!</span>
        </div>
    </div>
 </div>

Banners with Action

wifi_off You have lost the internet connection. This application is offline. 

<div class="banner">
    <div class="banner-container">
        <div class="banner-content">
            <i class="material-icons icon">wifi_off</i>
            <span>You have lost the internet connection. This application is offline.</span>
        </div>
        <div class="banner-actions">
            <button type="button" class="btn">action 1</button>
            <button type="button" class="btn">action 2</button>
        </div>
    </div>
 </div>

Classes Details

CSS Class Explanation
banner Required!
banner-container Container for banner content and actions
banner-content Contains banner text and icon
banner-actions Contains action buttons
