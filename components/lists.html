Lists with Form Control

You can easily add checkboxes, radio buttons, and switches to either start or end of list items.

  • Mobile Data Turn on mobile data such as email, and web
  • Enhanced LTE Services Turn on HD Voice and advanced communication
  • Mobile Data Turn on mobile data such as email, and web

// For Checkbox
 <ul class="list list--two-line">

            <span class="list-item-subtitle">Turn on mobile data such as email, and web</span>

Classes Details
CSS Class Explanation
list Container for all list content
list-item Contains icon, avatar, text and list actions
list-item-start Leading icon or avatar
list-item-detail Text wrapper
list-item-title, .title List primary text
list-item-subtitle, .subtitle1, .subtitle2 List secondary text
list-item-end List actions
is-active Activates list item
is-disabled Disables list item
list--dense Dense list
list--drawer Styling for navigation drawer list
list--two-line Double line list
list--three-line Triple line list
